# Schedule Creation Site

Локальный сайт для создания учебного расписания.

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

Откройте в браузере:

```text
http://127.0.0.1:5000
```

## Что умеет сайт

- загрузка Excel-файла по шаблону;
- проверка, что файл соответствует шаблону;
- ручной ввод аудиторий, преподавателей, предметов и групп без ручного ввода id;
- автоматическое создание расписания;
- просмотр расписания таблицей на сайте;
- проверка конфликтов;
- скачивание расписания в форматах ICS, Excel и PDF.

## Экспорт

В Excel и PDF создаётся общий лист/раздел расписания и отдельный лист/раздел для каждой группы.

## Шаблон

Пустой шаблон можно скачать на второй странице сайта кнопкой **Download template**.
